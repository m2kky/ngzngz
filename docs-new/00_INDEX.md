## مصطلحات سريعة
- Workspace: مساحة عمل (وكالة/فريق)
- Client: عميل
- Project: مشروع
- Task: مهمة
- Record: عنصر داخل قاعدة بيانات (صفحة/سجل)
- Database View: طريقة عرض محفوظة (Table/Board/Calendar…)
- Inbox: مركز التنبيهات + المنشنز + المهام المطلوبة منك
- Chat: محادثات الفريق داخل الـ Workspace
- AI Agent: مساعد ذكي ينفّذ أوامر داخل النظام حسب الصلاحيات
- Record Sheet: تفاصيل العنصر في Panel/Sheet
- Record Page: تفاصيل العنصر في صفحة كاملة
