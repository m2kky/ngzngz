# MVP Roadmap

## MVP-1 (Core Work Management)
- Onboarding (Role + Workspace)
- Clients + Projects + Tasks + Meetings
- Record Page موحّد (Sheet/Desktop + Full page/Mobile)
- Properties System (System + Custom basics)
- Views: Table/Board/Calendar + Filters + Saved Views
- Inbox (Notifications center) MVP
- Chat (Workspace chat) MVP
- Dojo (MVP-light): XP + streak + achievements بسيطة
- RBAC (Owner/Admin/Member/Guest كبداية)

## MVP-2 (Notion-like Power)
- Strategy Hub داخل العميل + index
- Brand Kit داخل العميل + index
- Automations MVP (Task triggers + notifications + create task)
- Saved Views: Shared + Default management

## MVP-3 (AI Assistant & Integrations)
- AI Chat (help/summary/suggestions)
- AI Agent (Read-only + Create/Update safe ops) + Audit log + Approvals
- Integrations: Meta (أولاً)
- Ads read-only + KPIs + client-scoped views
- Gmail/Drive basic linking (عرض/ربط)

## MVP-4 (Scale)
- Rollups/Formulas
- Ads controls + alerts متقدمة
- HR module (اختياري حسب الأولوية)
- Advanced AI policies (per role/per workspace/per module)
