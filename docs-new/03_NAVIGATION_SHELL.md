# Navigation Shell (Desktop vs Mobile)

## Desktop
- Sidebar (قابلة للطي) + Header
- أقسام أساسية:
  - Dashboard
  - Clients
  - Projects
  - Tasks
  - Meetings (Calendar)
  - Ads
  - Strategy Hub
  - Brand Kit
  - Automations
  - Settings
- سلوك:
  - الرجوع يحافظ على آخر View + Filters + scroll position
  - فتح Record افتراضيًا Sheet جانبي + زر Open as page

## Mobile
- Bottom Bar ثابت: 5 Tabs فقط
  - Home
  - Clients
  - Tasks
  - Calendar
  - More
- More (Bottom Sheet):
  - Projects, Ads, Strategy, Brand, Automations, Settings, Integrations
- FAB (زر +):
  - Create Task / Meeting / Project / Client
- سلوك:
  - Record يفتح Full Page افتراضيًا
  - Quick edit ممكن Bottom Sheet لتعديل 2-3 properties (Status/Assignee/Due)

## قاعدة ذهبية
نفس المسارات والصفحات (Deep links) على الموبايل والديسكتوب؛ الاختلاف في الـ shell فقط.
