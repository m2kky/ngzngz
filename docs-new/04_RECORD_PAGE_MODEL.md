# Record Page Model (Notion-like)

أي كيان (Client/Project/Task/Meeting/Strategy/Brand Asset/Ads Item) له نفس النموذج:

## 1) Header
- Icon + Title (Editable)
- Breadcrumbs (Workspace → Client → Project → …)
- Actions: Share / Duplicate / Archive / Open as page

## 2) Properties Row
- System properties (افتراضية حسب نوع الكيان)
- Custom properties (قابلة للإضافة/الإزالة/إعادة الترتيب)
- Add property → يفتح Property Manager

## 3) Body
- Sections/Blocks حسب نوع الكيان (Templates)
  - Task: Brief / Checklist / Deliverables / Links / Attachments
  - Project: Scope / Milestones / Timeline / Resources
  - Meeting: Agenda / Notes / Decisions / Action Items
  - Client: Overview + Shortcuts لتبويبات Strategy/Brand/Ads
- Embedded Database Views:
  - داخل Client: Projects/Tasks/Meetings مفلترة تلقائيًا على هذا العميل
  - داخل Project: Tasks/Meetings مفلترة تلقائيًا على هذا المشروع

## 4) Comments + Activity
- Comments thread
- Activity log (مين غيّر إيه وإمتى)

## فتح الـ Record
- Desktop: Sheet جانبي افتراضيًا + Open as page
- Mobile: Full page افتراضيًا
