# Vision & Scope

## الهدف
منصة لإدارة وكالات التسويق: تشغيل يومي + تخطيط + تنفيذ + متابعة إعلانات + تكاملات + أتمتة + صلاحيات.

## الهيكل الأساسي
- Workspace
  - Clients
    - Strategy Hub (SOSTAC)
    - Brand Kit
    - Ads Management (لكل عميل)
    - Projects
      - Tasks
    - Meetings
  - Dojo (XP + تحفيز)
  - Automations (Workspace-level)
  - Users & Roles (RBAC)
  - Client Portal (مراجعات العميل)

## المبادئ
- تجربة Notion-like:
  - كل كيان = Record Page موحّد (Properties + Body + Comments/Activity)
  - فتح سريع في Sheet جانبي (Desktop) أو صفحة كاملة (Mobile)
  - Views/Filters قوية + Saved Views
- Mobile-first navigation:
  - Mobile: Bottom Bar + More
  - Desktop: Sidebar
- Role-based:
  - كل صفحة/أكشن محكومة بالدور
- تحفيز من غير ما يبقى “لعبة”:
  - XP على سلوكيات شغل واضحة (إنهاء مهام/التزام بالمواعيد/تعاون)
  - Streaks وAchievements علشان الاستمرارية
  - Leaderboard اختياري حسب سياسة الـ Workspace

## خارج النطاق (حاليًا)

- White labeling الكامل
